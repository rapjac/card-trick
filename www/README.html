27 Cards Trick

Intro

No sleight of hand, no trick decks. Just math, combined with the power of the modern web. Built in HTML5 and CSS3 and powered by Backbone.js. Inspired by numberphile's Beautiful Card Trick

Libraries Used:
  • jQuery
  • Underscore.js
  • Backbone.js
  • Bootstrap

Getting Started

To run this web app, open index.html in the www directory

OR

while in the project root, run harp server and open localhost:9000 in your browser.
